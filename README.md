## Setup

1. Install [SuperCollider]

2. Add the `scsynth` binary to your `PATH`

3. Clone this repo and run the project:

   ~~~ bash
   $ git clone https://github.com/underscoreio/compose.git

   $ cd compose

   $ ./sbt.sh run
   ~~~
